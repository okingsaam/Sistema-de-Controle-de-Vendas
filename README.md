# Sistema de Controle de Produtos em Java

## 📌 Descrição

Este é um sistema simples de controle de produtos desenvolvido em **Java**, executado via terminal.
O projeto foi criado com o objetivo de praticar **Programação Orientada a Objetos (POO)**, organização em camadas e operações básicas de gerenciamento de dados.

O sistema permite cadastrar, listar, atualizar, buscar e remover produtos, além de calcular o valor total do estoque.

---

## 🚀 Funcionalidades

* 📦 Cadastro de produtos
* 📋 Listagem de produtos
* ✏️ Atualização de produtos
* ❌ Remoção de produtos
* 🔍 Busca de produto por ID
* 💰 Cálculo do valor total do estoque

---

## 🛠 Tecnologias utilizadas

* Java
* IntelliJ IDEA
* Programação Orientada a Objetos (POO)
* Estrutura em camadas (Model / Service)

## ▶ Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/seu-repositorio.git
```

2. Abra o projeto no **IntelliJ IDEA**

3. Execute o arquivo:

```
Main.java
```

4. Utilize o menu no terminal para interagir com o sistema.
